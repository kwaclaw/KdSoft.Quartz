Class CronTriggerUpdate

Trigger update. Subset of CronTriggerInfo. Includes additional CronExpressionString and TimeZoneId properties for updating CRON triggers. Any KdSoft.Utils.ValueWrapper<T> property left null will be ignored for the update.
