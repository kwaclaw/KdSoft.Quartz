JobRunShell

JobRunShell instances are responsible for providing the 'safe' environment for IJob s to run in, and for performing all of the work of executing the IJob, catching ANY thrown exceptions, updating the ITrigger with the IJob's completion code, etc.

A JobRunShell instance is created by a IJobRunShellFactory on behalf of the QuartzSchedulerThread which then runs the shell in a thread from the configured System.Threading.ThreadPool when the scheduler determines that a IJob has been triggered.
