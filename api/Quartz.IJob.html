Interface IJob

The interface to be implemented by classes which represent a 'job' to be performed.

Namespace: Quartz
Assembly: Quartz.dll
Syntax
public interface IJob
Remarks

Instances of this interface must have a langword_csharp_public no-argument constructor. JobDataMap provides a mechanism for 'instance member data' that may be required by some implementations of this interface.

Methods

Execute(IJobExecutionContext)

Called by the IScheduler when a ITrigger fires that is associated with the IJob.

Declaration
void Execute(IJobExecutionContext context)
Parameters
Type Name Description
IJobExecutionContext context

The execution context.
Remarks

The implementation may wish to set a result object on the JobExecutionContext before this method exits. The result itself is meaningless to Quartz, but may be informative to IJobListeners or ITriggerListeners that are watching the job's execution.
